Overview:

This project demonstrates an end-to-end data analytics workflow, covering data extraction, analysis, visualization, and reporting. 
The goal is to transform raw transactional data into actionable business insights using industry-standard tools commonly used by data analysts.

The project includes Python-based data analysis, SQL querying on a MySQL server, interactive dashboard development in Power BI, 
and final reporting through written documentation and presentation slides.
________________________________________
Dataset:

o	Type: Customer transaction / shopping behavior data

o	Records: ~3,900 rows

o	Features:
o	Customer demographics (age, gender, location, subscription status)

o	Purchase details (category, item, amount, season, discounts)

o	Behavioral metrics (purchase frequency, reviews, shipping type)

The dataset contains minor missing values that were handled during data cleaning.
________________________________________
Tools & Technologies

o	Python: Data loading, cleaning, and exploratory data analysis

o	Libraries: pandas, numpy, matplotlib, seaborn

o	SQL (MySQL): Business analysis using structured queries

o	Power BI: Interactive dashboard and visual storytelling

o	Gamma: Presentation (PPT) creation

o	Jupyter Notebook: Python analysis environment
________________________________________
Project Steps:-

1.	Data Loading
o	Imported the dataset into Python using pandas.

2.	Exploratory Data Analysis (EDA)
o	Reviewed dataset structure and summary statistics.
o	Identified trends, outliers, and distributions.
o	Analyzed customer demographics and purchase behavior.

3.	Data Cleaning & Preparation  
o	Handled missing values.
o	Standardized column names.
o	Performed feature engineering (age groups, customer segments).
o	Ensured data consistency.

4.	SQL Analysis (MySQL)   
o	Loaded cleaned data into a MySQL database.
o	Wrote SQL queries to answer business questions such as:
o  Revenue by customer segment
o	Top-performing products
o	Subscription vs non-subscription behavior
o	Discount and shipping impact on spending

5.	Power BI Dashboard   
o	Built an interactive dashboard for business users.
o	Included KPIs, trends, and filters for easy exploration.

6.	Reporting & Presentation
o	Created a written report summarizing findings.
o	Designed a presentation (PPT) in Gamma for stakeholders.
________________________________________
Dashboard

o	Top categories and products

o	Subscription and discount insights

o	Revenue trends by demographics

The dashboard enables dynamic filtering and supports data-driven decision making.

Results & Key Insights
